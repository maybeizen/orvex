# Orvex

Monorepo for Orvex, a hosting company.

### `/api` - Express-based Backend server. (Probably will switch to Go or Rust)

- Typescript
- Express
- MongoDB

### `/frontend` - Next.js Frontend server.

- Typescript (TSX)
- React
- Next.js

### `/bot` - Discord.js + Javascript Discord bot for the Orvex Discord server.

- Javascript
- Discord.js
- MongoDB
